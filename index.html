My Online Interactive Resume

This is my second project of Udacity's Front-End Nanodegree (the first is actually this portfolio site! =) ) and is an online interactive resume. This was the first project where I started using Javascript, learned how to use JS syntax, implement jQuery, and did some simple DOM manipulation. I used Google Maps API to display the various locations where I've lived and worked.

Classic Arcade Game

In this project, I learned the fundamentals of basic game design by creating a tile-based game that is playable on the browser. The game engine and image assets were provided and I used object-oriented Javascript to render the game's map and objects as well as add functionality. Although this project fulfilled and exceeded Udacity's Front-End Nanodegree requirements, I have two additional levels designed that will be added to the game in the near future.

Website Optimization

In this project, I was given a sample website that contained links to other additional pages. The objective was to perform website optimization techniques to get the page to load at 60fps and also gain Google PageSpeed Insights scores of above 90% on both mobile and desktop platforms. I used grunt with node.js to compress the css code and also compressed the sizes of the images to make the load times faster.

Five Awesome Chinese Restaurants in San Francisco, CA

This is my fifth project of Udacity's Front-End Nanodegree and it is currently a work in progress. In this project, the goal is to create a webapp using Google Maps API to create a map displaying five locations of interest (I have chosen to use five highly reviewed chinese restaurants in San Francisco, CA), along with a listview (using knockout.js), a search filter, and the use of a third party API. In my project, I will be using Yelp's API to display reviews/ratings of the restaurants that will display both in the listview and in the location marker infowindows on the map.
